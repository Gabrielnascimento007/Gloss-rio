Desafios HTML

HTML
O elemento HTML (ou HTML root element) representa a raiz de um HTML documento. Todos os outros elementos devem ser descendentes desse elemento.


Main
O elemento define o conteúdo principal dentro do em seu documento ou aplicação


Section
O elemento HTML representa uma seção genérica contida em um documento HTML,


Header
O elemento HTML representa um grupo de suporte introdutório ou navegacional.
